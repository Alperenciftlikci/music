# Music AI — Website

Static landing + legal pages for the **Music AI** iOS app.
Hosted on GitHub Pages at <https://alperenciftlikci.github.io/music/>.

## Pages
- `index.html` — landing page
- `privacy.html` — Privacy Policy (App Store required)
- `terms.html` — Terms of Use (auto-redirects to Apple's Standard EULA)
- `support.html` — Support / FAQ

## Local preview
```bash
python3 -m http.server 8000
# open http://localhost:8000
```
